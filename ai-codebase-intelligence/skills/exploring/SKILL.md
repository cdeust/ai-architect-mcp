# Exploring Skill

## Purpose
Navigate and understand unfamiliar codebases systematically using graph-powered intelligence.

## When to Use
- First encounter with a new repository
- Understanding module boundaries and architecture
- Finding entry points and key abstractions

## Operations

1. **Index**: Run `ai_architect_codebase_analyze` to build the knowledge graph
2. **Map**: Query for high-level structure — communities, processes, entry points
3. **Navigate**: Use `ai_architect_codebase_context` for 360° views of key symbols
4. **Trace**: Follow execution flows via process membership

## Workflow

```
Index repo → List communities → Pick community → Explore members → Trace processes
```

## Output
- Module map with cohesion scores
- Key processes with entry points
- Dependency summary between communities
